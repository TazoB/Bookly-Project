# ­ЪЊџ Bookly РђЊ рЃгрЃўрЃњрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ JavaFX-рЃўрЃЌ

**Bookly** рЃљрЃарЃўрЃА рЃЊрЃћрЃАрЃЎрЃбрЃЮрЃъ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў Java рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџ рЃћрЃюрЃљрЃќрЃћ. рЃўрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃАрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃА рЃЏрЃљрЃарЃбрЃўрЃЋ рЃЊрЃљ рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџ рЃњрЃќрЃљрЃА рЃгрЃўрЃњрЃюрЃћрЃЉрЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃљрЃЦрЃбрЃўрЃЋрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ.

## ­ЪќЦ№ИЈ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ?

рЃЌрЃЦрЃЋрЃћрЃю рЃњрЃљрЃЦрЃЋрЃЌ рЃЮрЃарЃў рЃњрЃќрЃљ:

### 1. Git-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ  
рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃљрЃЏ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА Git repository-рЃў рЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЉрЃљ рЃЏрЃЮрЃљрЃ«рЃЊрЃўрЃюрЃЮрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃерЃў рЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃљрЃЌ рЃЎрЃЮрЃЊрЃў Java-рЃА рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃЎрЃЮрЃЏрЃърЃљрЃўрЃџрЃћрЃарЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃЌ, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ **IntelliJ IDEA**.

### 2. Executable рЃцрЃљрЃўрЃџрЃўрЃЌ (рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ)  
рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЊрЃљрЃ«рЃЋрЃўрЃЊрЃћрЃЌ [Google Drive рЃџрЃўрЃюрЃЎрЃќрЃћ](https://drive.google.com/drive/folders/1SgVv0NT-YM3vfZjV6WyL8s65TGHO2nUe?usp=sharing) рЃЊрЃљ рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃЮрЃЌ **Bookly-рЃўрЃА рЃцрЃЮрЃџрЃЊрЃћрЃарЃў**.  
рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃАрЃљрЃерЃЋрЃћрЃЉрЃљрЃЊ рЃЮрЃарЃ»рЃћрЃа рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ `Bookly.exe` рЃцрЃљрЃўрЃџрЃќрЃћ.
